Snippets in MadCap Flare

A snippet (file with .flsnp extension) is a piece of content that can be used repeatedly in multiple HTM files. Its content can be formatted the same way as any other HTM file and it can contain any type of HTML element (text, images, tables).

By using snippets, you have the possibility to change the content in one place. Each HTM file using this snippet will have the same content automatically. This makes your documentation easier to update and avoid having outdated information.

NOTE: The content of a snippet file is the same as in an HTM file. However, when inserting a snippet to an HTM file only its <body> content is added.

You can enable snippet auto suggestion so that Flare will suggest a snippet that matches the text you are typing. To enable it, go to File > Options > Auto Suggestion.

Create a Snippet File

  1. In the Content Explorer, right-click on any folder.
  2. Go to New > Snippet.
  3. Specify the source of the file (either from template or another existing snippet).
  4. Type the snippet file name in the File Name field.
  5. (Optional) Specify the destination folder in the Folder field.

OR

  1. Select existing content in an HTM file.
  2. Go to the Home ribbon.
  3. Click the Create Snippet button.
  4. Type the snippet file name in the Snippet File field.
  5. Select the destination folder in the Project Folder field.

Insert Snippet

  1. Place the text cursor in the HTM file where you want to place the snippet.
  2. Click the Insert a snippet button either in the Insert ribbon or in the XML Editor toolbar.
  3. Select the snippet from the project structure.

Additional Actions

Select New Snippet

  1. Right-click on a snippet in the XML Editor.
  2. Choose Select New Snippet.
  3. Select a new snippet from the project structure.

Convert Snippet to Text

  1. Right-click on a snippet in the XML Editor.
  2. Select Convert to text.

Edit Snippet

  1. Right-click on a snippet in the XML Editor.
  2. Select Open Snippet File.

OR

  1. In the Content Explorer, navigate to the folder where the snippet file is located.
  2. Double-click on the snippet file.

CAFFEINE Functionality

The CAFFEINE feature increases the overall efficiency of your company's employees. It has both short- and long-term properties. CAFFEINE should be introduced to each employee in liquid state called coffee. It has the best impact at the beginning of the work shift. The maximum amount of CAFFEINE per employee varies and is dependent on the current tasks of said employee. As a rule of thumb, the more tasks an employee has, the more CAFFEINE should be introduced.

IMPORTANT: Once CAFFEINE has been introduced to your company, remember to keep a one-week supply buffer in storage, as when employees will be cut off from the coffee supply their efficiency will drop dramatically.

How to Enable CAFFEINE

  1. Buy the most expensive espresso machines as Primary (one per 20 employees).
  2. Buy at least two more as Backups.
  3. Buy non-grounded coffee supply (0.5 kg per employee).
  4. Install the espresso machines evenly around the workspace.
  5. Stick ~20 flyers around the espresso machine containing different procedures and rules on how to use the machine.
  6. Hire a maintenance crew and station them as close to the espresso machines.
  7. Setup a rule in Outlook to automatically forward any e-mail containing words like coffee or espresso to the maintenance crew for quick response in case of an espresso machine failure.
